# LLM Fundamentals: Tokenization, Embeddings and Prompt Engineering

This repository provides a comprehensive introduction to the fundamental concepts behind Modern Large Language Models (LLMs). The project was developed as part of the Generative AI Internship at the National Centre of Artificial Intelligence (NCAI), UET Lahore.

Instead of training a language model, this project focuses on understanding how modern Transformer-based models process text before inference or fine-tuning.

---

## Project Objectives

The objective of this project is to understand and demonstrate the core building blocks of Large Language Models (LLMs), including text tokenization, embeddings, positional encoding, Transformer inputs, and prompt engineering techniques.

The notebook combines theoretical explanations with practical demonstrations using Hugging Face Transformers.
